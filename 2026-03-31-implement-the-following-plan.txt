  # Map Theme Selection — Implementation Plan                               
                                                                            
  ## Overview                                                               
  Add map theme/style selection to both Map screen and Pick Location        
  screen, with persistence via Zustand store and visual theme selector in   
  Settings.                                                                 
                                                                            
  **5 themes:** Clean (current default), Dark, Nature, Satellite, Terrain   
                                                                            
  ---                                                                       
                                                                            
  ## Files to Create                                                        
                                                                            
  ### 1. `src/constants/mapStyles.ts` — Theme definitions (source of truth) 
  - Export `MapThemeId` type: `'clean' | 'dark' | 'nature' | 'satellite' |  
  'terrain'`                                                                
  - Export `MAP_THEMES` array with each theme's metadata:                   
  - `id`, `icon` (Ionicons), `previewColor` (for UI card bg), `mapType`     
  prop, `customMapStyle` array                                              
  - **Clean**: current `CUSTOM_MAP_STYLE` (hides POIs, subtle colors) +     
  `mapType: 'standard'`                                                     
  - **Dark**: dark Google Maps JSON style (dark roads, dark landscape, blue 
  water) + `mapType: 'standard'`                                            
  - **Nature**: green-tinted style (green landscape, blue water, earthy     
  roads) + `mapType: 'standard'`                                            
  - **Satellite**: `mapType: 'satellite'`, no `customMapStyle`              
  - **Terrain**: `mapType: 'terrain'` (Android) / `'mutedStandard'` (iOS),  
  no `customMapStyle`                                                       
  - Export helper `getMapTheme(id: MapThemeId)` that returns the theme      
  config                                                                    
                                                                            
  ### 2. `src/stores/mapStore.ts` — Zustand store with AsyncStorage         
  - Follow existing `authStore.ts` pattern                                  
  - State: `{ mapTheme: MapThemeId }`                                       
  - Actions: `setMapTheme(id)` — updates state + persists to AsyncStorage   
  - Action: `loadStoredPreferences()` — reads from AsyncStorage on app      
  start                                                                     
  - Default: `'clean'`                                                      
                                                                            
  ### 3. `src/app/settings/components/MapThemeSelector.tsx` — Visual theme  
  picker                                                                    
  - Horizontal row of small theme cards (similar to language selector       
  pattern)                                                                  
  - Each card: colored preview box + label + checkmark if selected          
  - Props: none (reads/writes directly from `useMapStore`)                  
  - Uses `t('settings.mapThemeClean')` etc. for labels                      
                                                                            
  ---                                                                       
                                                                            
  ## Files to Modify                                                        
                                                                            
  ### 4. `src/stores/index.ts`                                              
  - Add export: `export { useMapStore } from './mapStore'`                  
                                                                            
  ### 5. `src/app/settings/index.tsx` — Add theme selector to Map           
  Preferences                                                               
  - Import `MapThemeSelector`                                               
  - Insert `<MapThemeSelector />` inside the Map Preferences card (after    
  existing toggles)                                                         
                                                                            
  ### 6. `src/app/(tabs)/map.tsx` — Use theme from store

  ### 7. `src/app/pick-location.tsx` — Use theme from store

  - Import `useMapStore` + `getMapTheme`

  ### 8. `src/app/_layout.tsx` — Load stored preferences on app start
